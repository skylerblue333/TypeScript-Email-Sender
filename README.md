# TypeScript-Email-Sender

![CI](https://github.com/skylerblue333/TypeScript-Email-Sender/workflows/CI/badge.svg)

Production-ready TypeScript Express API for sender operations.

## Features
- Fully typed Express handlers
- Supertest integration testing
- Docker containerization
- GitHub Actions CI Pipeline

## Quick Start
```bash
npm ci
npm test
npm run build && npm start
```
